# Quiz - T-010

1. What is the canonical collection name for ingested messages?

<details>
<summary>Answer</summary>
`Messages` (Mongo collection `messages`).
</details>

2. Which field has a unique index to prevent duplicates?

<details>
<summary>Answer</summary>
`id`.
</details>

3. What are the allowed `source` values for canonical messages?

<details>
<summary>Answer</summary>
`osx_messages_app` and `sim_router`.
</details>

4. What are the allowed `phoneNumberId` values?

<details>
<summary>Answer</summary>
`primary` and `fallback`.
</details>

5. Which status values are defined for message lifecycle state?

<details>
<summary>Answer</summary>
`raw`, `imported`, `approved`, `hidden`, and `flagged`.
</details>

6. Which non-unique indexes support common query paths in T-010?

<details>
<summary>Answer</summary>
Indexes on `receivedAt`, `status`, and `source`.
</details>
