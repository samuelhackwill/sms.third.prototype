# Quiz - T-050A

1. Which file is used as the client main module for route bootstrapping?

<details>
<summary>Answer</summary>
`client/routes.js` via `package.json` -> `meteor.mainModule.client`.
</details>

2. Which two routes were added for the stage/admin skeleton?

<details>
<summary>Answer</summary>
`/stage` and `/admin/stage`.
</details>

3. In the tryhard-style approach, how are templates rendered by routes?

<details>
<summary>Answer</summary>
Directly with `this.render('<templateName>')` inside each FlowRouter route action.
</details>

4. What is the BroadcastChannel name shared by admin and stage pages?

<details>
<summary>Answer</summary>
`stage_test`.
</details>

5. Which template includes the placeholder mount node for future Pixi work?

<details>
<summary>Answer</summary>
The `stage` template, with `<div id="stageCanvasHost"></div>`.
</details>

6. What payload is sent when pressing Send 5?

<details>
<summary>Answer</summary>
`{ type: "spawn", messages: makeFakeMessages(5) }`.
</details>
